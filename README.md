# Full Stack Web Application

## Description

This is a full-stack web application with **Node.js** as the backend and **Next.js** as the frontend.  
The project follows a modern development workflow with a clean and scalable architecture.

---

## Tech Stack

### Frontend

- Next.js (React)
- JavaScript
- HTML / CSS

### Backend

- Node.js
- Express.js

### Database

- MySQL

---

## Database

The application uses **MySQL** as the primary database engine.  
The database is managed locally using **DBngin** for development purposes.

---

## Development Tools Used

### DBngin

DBngin is used to run and manage the MySQL database locally.  
It helps in handling database connections and data storage during development.

### Herd

Herd is used as the local development environment to run the backend and frontend services smoothly.
